Why coding still matters in the AI age: Tools like ChatGPT and Google Gemini can write code for you — but to use them powerfully (what people call vibe coding), you need to understand what to ask, why it works, and how to verify the output. A solid programming foundation turns you from a passive user into a confident problem-solver who gets real results from AI.

About This Course

This Python course is designed for absolute beginners who want to learn the core concepts of programming. The emphasis is on learning how to think like a programmer, not just memorising syntax. By the end, you will have the confidence to write basic programs, debug errors, read documentation, and tackle real problems — with or without AI assistance.

The course includes 100+ practice questions and online assistance to help you solidify your understanding. Recordings of all live sessions will be available.
